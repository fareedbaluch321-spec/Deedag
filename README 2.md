# Deedag - Book Sharing Platform

📚 **کتابوں کی شیئرنگ اور کرائے کا منصہ**

## خصوصیات

✅ کتابیں شیئر کریں  
✅ کتابیں کرائے پر لیں  
✅ دوسری کتابوں کو تلاش کریں  
✅ اپنی کتابوں کو منیج کریں  
✅ صارف دوست انٹرفیس  
✅ موبائل کے لیے تیار (Responsive)

## ٹیکنالوجی

- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Responsive Grid Layout
- **Compatibility**: تمام جدید براؤزر

## شروع کریں

1. **Repository Clone کریں**
   ```bash
   git clone https://github.com/fareedbaluch321-spec/deedag-baloch-book-share.git
   cd deedag-baloch-book-share
   ```

2. **فائل کھولیں**
   `index.html` کو اپنے براؤزر میں کھولیں

3. **مزہ لیں** 📖

## استعمال

### کتاب شیئر کریں
- "اپنی کتاب شیئر کریں" سیکشن میں جائیں
- تمام معلومات درج کریں
- "شیئر کریں" پر کلک کریں

### کتاب کرائے پر لیں
- "کرائے پر کتاب لیں" سیکشن میں جائیں
- کتاب کی تفصیلات درج کریں
- "درخواست بھیجیں" پر کلک کریں

### کتاب تلاش کریں
- سرچ بار میں کتاب کا نام یا مصنف درج کریں
- نتائج خود بخود فلٹر ہوں گے
